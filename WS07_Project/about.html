About Nature Journal

Nature Journal is a demo site built as a student project to practice responsive layout, semantic HTML, and accessible navigation.

Purpose

This small site demonstrates:

  • Responsive design using CSS Grid and Flexbox
  • Consistent navigation across pages
  • Lightweight, standards-friendly markup

How to use

Open the pages and resize your browser or use developer tools to simulate mobile widths. The navigation collapses into a menu button on small screens.
